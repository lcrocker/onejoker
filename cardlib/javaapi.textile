h1. Using the library in your Java code


h3. Java

Your Java programs will have to import the classes, and also load the library. More details of the classes and functions available are in the ["Java API":/onejoker/cardlib/javaapi.html] document. Here's a short example:

{% highlight java %}
import com.onejoker.cardlib.*;

public class Hello {
    static {
        System.loadLibrary("ojcard");
    }
    public static void main(String[] args) {
        CardList deck = new CardList(52), hand = new CardList(5);
        deck.fill();
        deck.shuffle();

        for (int i = 0; i < 5; ++i) hand.add(deck.pop());
        System.out.println(hand);
    }
}
{% endhighlight %}
